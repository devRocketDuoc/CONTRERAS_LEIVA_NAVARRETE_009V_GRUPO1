# Docs

Documentacion base del proyecto Air-Fly-Trip.

## Arquitectura activa

El sistema opera con 10 microservicios activos:

- `auth-service`
- `terminal-service`
- `vehicle-service`
- `route-service`
- `trip-service`
- `reservation-service`
- `tariff-service`
- `payment-service`
- `notification-service`
- `charging-station-service`
